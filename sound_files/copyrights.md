# 정리 규칙
파일 명은
Type_number_explanation 으로 정리합니다.

Type: 효과음의 경우 e (effect) / 음악 혹은 지속적으로 플레이되는 배경소리의 경우 b (background)로 표기
number: 실제 개발 시 헷갈리지 않도록 모든 소리파일에 넘버링을 부여합니다.
explanation: 함수명과 같이 이 음악파일을 설명하는 가벼운 타이틀입니다.

예시)
e_13_click : 효과음₁3번째_클릭 효과음 이라는 뜻 입니다.
